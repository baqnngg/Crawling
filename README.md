# 📍 Naver Map 음식점 크롤러

이 프로젝트는 **네이버 지도**에서 검색 결과를 자동으로 스크롤하고, 음식점 정보를 추출하여 **엑셀 파일로 저장**해주는 Python 크롤러입니다.

---

## ✅ 주요 기능

- 네이버 지도에서 음식점 검색 결과 자동 크롤링
- 다중 페이지 자동 스크롤 및 페이지 이동
- 이름, 카테고리, 설명, 별점, 리뷰 수 등 정보 수집
- 결과를 Excel(`.xlsx`) 파일로 저장
